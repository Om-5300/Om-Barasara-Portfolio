# My Personal Portfolio

A modern, responsive portfolio website built with Next.js, Framer Motion, and Tailwind CSS.

## Features

- **Modern UI/UX**: Clean and interactive design.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Animations**: Smooth transitions and effects using Framer Motion.
- **Tech Stack**: Next.js, React, Tailwind CSS, Particles.js.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run the Development Server**:
   ```bash
   npm run dev
   ```

3. **Open the App**:
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

You can deploy this project using [Vercel](https://vercel.com/) or [Netlify](https://www.netlify.com/).
