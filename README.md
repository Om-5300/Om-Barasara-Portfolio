# Om Barasara | Full-Stack Developer & AI Specialist

A modern, high-performance portfolio website built with **Next.js**, **Framer Motion**, and **Standard CSS**. This project showcases my journey, technical skills, and featured projects including AI-powered automation agents and luxury e-commerce platforms.

![Portfolio Preview](.github/images/img_main.png)

## 🚀 Key Features

-   **Instant Navigation**: Optimized route prefetching and dynamic importing for a lag-free experience.
-   **Intelligent UI/UX**: Professional "Pill" navigation for desktop and an animated "Burger Menu" for mobile/tablet.
-   **Fully Responsive**: Meticulously designed for phones, tablets, and high-resolution desktops using standard CSS Grid and Flexbox.
-   **Dynamic Animations**: Smooth, high-speed page transitions and background particle effects using Framer Motion and tsparticles.
-   **AI Focused**: Dedicated sections showcasing RAG systems, LLM agents, and automation workflows.

## 🛠️ Tech Stack

-   **Frontend**: Next.js 15, React 18, Framer Motion
-   **Styling**: Standard CSS (Custom Properties), Tailwind CSS (Base)
-   **Animations**: tsparticles, Swiper.js
-   **Deployment**: Vercel / Netlify

## 🏁 Getting Started

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Om-5300/Om-Barasara-Portfolio.git
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```
